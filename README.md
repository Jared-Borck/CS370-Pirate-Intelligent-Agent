# CS370-Pirate-Intelligent-Agent


## Project Reflection

For this project, I worked on a pirate intelligent agent that used reinforcement learning to solve a treasure hunt maze problem. The goal was for the pirate agent to learn how to move through the environment and find the treasure. I was given starter code that included the basic environment, the maze setup, the experience replay structure, and some of the neural network support. The given code helped define how the game worked and how the agent could interact with the maze.

The main code I created myself was the deep Q-learning training logic for the intelligent agent. I completed the part of the notebook that trained the pirate to choose actions, explore the environment, store experiences, and improve over time. The agent learned by trying different moves, receiving rewards or penalties, and using those results to make better decisions in future episodes. This project helped me understand how reinforcement learning can be used to solve a pathfinding problem without directly hardcoding every step.

Computer scientists solve problems by designing, building, testing, and improving systems. This matters because computer science is used in many areas, including artificial intelligence, cybersecurity, healthcare, business, education, and everyday technology. In this project, computer science mattered because the intelligent agent had to use data, algorithms, and decision-making to solve a problem that could not be handled by a simple fixed set of instructions.

When I approach a problem as a computer scientist, I try to break the problem into smaller parts first. For this project, that meant understanding the environment, the possible actions, the reward system, and how the agent would learn from experience. I also had to test the code and watch how the agent improved over time. Breaking the problem down made it easier to understand what the code was doing and how the learning process worked.

My ethical responsibilities to the end user and the organization include creating systems that are reliable, fair, secure, and understandable. For AI projects, it is important to make sure the system is tested and does not make harmful or biased decisions. It is also important to protect user data and be clear about how an AI system works. Even though this project was a game-based maze problem, the same ethical ideas apply to real AI systems. Developers should make sure their systems support users, meet organizational goals, and avoid causing harm.

Overall, this project helped me better understand reinforcement learning, neural networks, and how intelligent agents can learn from experience. It also showed me that computer science is not just about writing code. It is about solving problems in a careful and responsible way.
